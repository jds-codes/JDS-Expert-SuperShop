# 🛒 JDS EXPERT - SuperShop Management System

A professional **ASP.NET Core MVC** web application for managing a modern SuperShop. 
Specifically features **Coffee, Fresh Juice, and Cookies**.

## ✨ Features
- **Modern UI:** Red & Black theme inspired by JDS EXPERT branding.
- **Product Management:** Full CRUD operations for shop items.
- **Dynamic Search:** Search functionality for products and categories.
- **Database:** SQL Server integration with Entity Framework Core.
- **Pre-loaded Items:** 9 starter products (Coffee, Juice, Cookies).

## 🛠️ Tech Stack
- C# / ASP.NET Core 8.0
- Entity Framework Core (SQL Server)
- Bootstrap 5 & Bootstrap Icons

## 🚀 How to Run
1. Clone the repository.
2. Update the connection string in `appsettings.json`.
3. Run `Update-Database` in Package Manager Console.
4. Press F5 to run.
